# Work with docker

## Images
https://hub.docker.com/_/python

## Start container
docker-compose up -d

## Stop container
docker-compose stop

## container name
To access the python container, you can use
`python`

To access the jupyter container, you can use
`jupyter-py`

To access the container via console:
`docker exec -it python bash`
